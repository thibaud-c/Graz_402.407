### Create a website from your github repository

Follow this [tutorial](https://docs.github.com/en/pages/getting-started-with-github-pages/creating-a-github-pages-site) to create a website from your GitHub repository

1. Settings -> pages 
2. Choose static website
3. Update the yaml file with the root of your project (index.html): `line 40: path: /folderName`

4. On the homepage of your repository look on the right to deployment -> you should have the url of your project
`username.github.io/folderName`
